# Array Introduction:

## what is an Array:
In C programming, an array is a collection of elements of the same data type stored in contiguous memory locations. It allows you to store and manage multiple values using a single variable name, with individual elements accessed via indices. 

to declare an array:
 ``` data_type name_of_array[size_of_array]```
## Example:
```
int arr[10];
```
## Assigning values in Array:
```
arr[index] = value;
arr[9]=10;
arr[2]=112;
```
## Print Array value:
```
printf("format_specifier",name_of_arr[index]);
```
